Inventário de Dados Essenciais

APÊNDICE AL: Ferramenta fundamental para o Pilar 1, permitindo identificar, organizar e avaliar a relevância dos dados operacionais e estratégicos da PME.

Este inventário permite identificar quais dados existem, onde estão armazenados, quem são seus responsáveis e qual sua relevância para o negócio, servindo como base para a padronização e melhoria da qualidade da informação.

AL.1 Instruções para Uso do Inventário

  • Mapeamento Inicial: O inventário deve ser preenchido inicialmente de forma colaborativa, envolvendo representantes de todas as áreas chave da PME (ver Apêndice D).
  • Detalhamento Progressivo: Comece listando os dados mais óbvios e críticos. O inventário é um documento vivo e pode ser expandido ao longo dos ciclos de GD.
  • Foco na Essência: O objetivo não é listar todos os dados, mas os "dados essenciais" para a operação e estratégia, especialmente aqueles com maior impacto ou risco.
  • Identificação de Problemas: A seção "Observações / Desafios" é crucial para registrar problemas de qualidade, acesso, segurança, etc., que servirão de insumo para o plano de ação.
  • Base para Outras Ações: Este inventário é a base para a padronização (Pilar 1), para a definição de políticas (Pilar 3) e para identificar necessidades de capacitação (Pilar 2).
  • Revisão Periódica: Recomenda-se que o inventário seja revisado e atualizado pelo menos semestralmente ou sempre que houver mudanças nos processos ou sistemas.

Fonte: Adaptado do Pilar 1 do modelo GD-PME 3 Pilares e em boas práticas de inventário de dados (DAMA INTERNATIONAL, 2017).
