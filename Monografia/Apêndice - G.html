Modelo de Termo de Boas Práticas em Dados (Simplificado)

APÊNDICE G: Documento institucional para formalizar políticas mínimas de governança de dados em PMEs, alinhado ao Pilar 3 do modelo GD-PME.

Este modelo visa garantir clareza, acessibilidade e compromisso institucional sem burocratização excessiva. É essencial para a execução do Pilar 3 (Políticas Graduais e Simplificadas) e fomenta a conscientização dos colaboradores (Pilar 2).

Empresa:

Data de Emissão:

1. Finalidade

  • Este documento estabelece diretrizes mínimas para o uso, armazenamento, atualização e compartilhamento de dados dentro da empresa, com base no modelo GD-PME 3 Pilares.

2. Regras Básicas

  • Toda informação relevante deve ser armazenada em repositório comum previamente definido;
  • É proibido alterar, excluir ou compartilhar dados sem autorização expressa da liderança;
  • Backups serão realizados semanalmente;
  • Dados sensíveis (ex.: CPF, e-mail, dados financeiros) exigem cuidado especial e acesso restrito.

3. Responsabilidades

  • A equipe está autorizada a reportar erros e sugerir melhorias;
  • Cabe à liderança zelar pela clareza e aplicação das diretrizes.
